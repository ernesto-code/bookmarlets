Clone element Instructions:
  1. Inspect the code and find the dynamically added element's container.
  2. Add/Modify its class attribute's value, ensuring it contains the "a11ytest" class.
  3. Open the browser console.
  4. Activate the "Clone element" bookmark.
  5. Check that "Observed and Cloned .... Done!" has been log in console.
  6. The dynamically added element must be already included in the DOM.
Show currently focused element when any key is pressed - Instructions:
  1. Activate the "Current focused element when keypress" bookmark.
  2. Open the browser console.
  3. When navigating using Tab key (or any key is pressed), the currently focused element will be log in the console.
Show currently focused element when "p" key is pressed - Instructions:
  1. Activate the "Current focused element (p)" bookmark.
  2. Open the browser console.
  3. When "p" key is pressed the currently focused element will be log in the console.
Current state - Instructions:
  1. Inspect the button to test.
  2. Add/Modify its class attribute's value, ensuring it contains the "a11ybutton" class.
  3. Open the browser console.
  4. Activate the "Current state" bookmark.
  5. Once the button is activated, the current state will be log in the console.

* Remember that any of these bookmarlets can be renamed once placed as bookmark.
